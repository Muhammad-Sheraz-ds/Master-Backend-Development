# 1. Why Functions?

Imagine you need to calculate the average marks of students in many places.

Without functions:

```javascript
let avg1 = (80 + 75 + 90) / 3;
let avg2 = (70 + 85 + 95) / 3;
let avg3 = (60 + 78 + 88) / 3;
```

This repeats the same logic.

Instead, create one function.

```javascript
function average(a, b, c) {
    return (a + b + c) / 3;
}
```

Now reuse it.

```javascript
console.log(average(80, 75, 90));
console.log(average(70, 85, 95));
```

# 2. Function Syntax

```javascript
function functionName(parameters) {

    // statements

    return value;

}
```

Example

```javascript
function square(number) {

    return number * number;

}
```

Calling

```javascript
console.log(square(5));
```

Output

```
25
```

---

# 3. Function Parameters

Parameters receive values from the caller.

```javascript
function greet(name) {

    console.log("Welcome " + name);

}

greet("Ali");
greet("Sara");
```

Output

```
Welcome Ali
Welcome Sara
```

---

# 4. Return Statement

The `return` statement sends a value back to the caller.

```javascript
function add(a, b) {

    return a + b;

}

let result = add(10, 20);

console.log(result);
```

Output

```
30
```

---

# 5. Default Parameters

JavaScript allows default values.

```javascript
function greet(name = "Guest") {

    console.log("Hello " + name);

}

greet();

greet("Ali");
```

Output

```
Hello Guest
Hello Ali
```

---

# 6. Arrow Functions

Modern JavaScript provides a shorter syntax.

Traditional

```javascript
function multiply(a, b) {

    return a * b;

}
```

Arrow

```javascript
const multiply = (a, b) => {

    return a * b;

};
```

Short version

```javascript
const square = x => x * x;
```

# 7. What is an Object?

An object groups related information together.

Example

```javascript
const student = {

    name: "Ali",

    rollNo: 101,

    cgpa: 3.85

};
```

Instead of storing separate variables,

```javascript
let name;
let rollNo;
let cgpa;
```

we store everything inside one object.

---

# 8. Accessing Object Properties

Dot notation

```javascript
console.log(student.name);
```

Bracket notation

```javascript
console.log(student["cgpa"]);
```

Output

```
Ali
3.85
```

---

# 9. Updating Objects

```javascript
student.cgpa = 3.95;

student.city = "Lahore";
```

Objects can grow by adding new properties.

---

# 10. Object Methods

Objects can also contain functions.

```javascript
const student = {

    name: "Ali",

    greet() {

        console.log("Welcome " + this.name);

    }

};

student.greet();
```

Output

```
Welcome Ali
```

---

# 11. Understanding `this`

Inside an object,

```javascript
this
```

refers to the current object.

```javascript
const car = {

    brand: "Toyota",

    show() {

        console.log(this.brand);

    }

};

car.show();
```

Output

```
Toyota
```

---

# 12. Passing Objects to Functions

```javascript
function printStudent(student) {

    console.log(student.name);

}

const s = {

    name: "Sara"

};

printStudent(s);
```

Objects are passed by sharing their reference.

Changes made to the object's properties inside the function are visible outside the function.

---

# 13. Practice Example

```javascript
const employee = {

    id: 101,

    name: "Ahmed",

    salary: 75000

};

function increaseSalary(emp, amount) {

    emp.salary += amount;

}

increaseSalary(employee, 5000);

console.log(employee.salary);
```

Output

```
80000
```

# Common Mistakes

❌ Forgetting `return`

```javascript
function add(a, b) {

    a + b;

}
```

Correct

```javascript
function add(a, b) {

    return a + b;

}
```

---

❌ Using `this` outside an object incorrectly.

---

❌ Forgetting to call the function.

Wrong

```javascript
square;
```

Correct

```javascript
square();
```
