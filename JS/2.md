# 1. Why Do We Need Arrays?

Imagine you want to store the marks of 100 students.

Without arrays:

```javascript
let marks1 = 80;
let marks2 = 76;
let marks3 = 95;
let marks4 = 65;
...
let marks100 = 88;
```

This is difficult to manage.

Arrays allow us to store many values under one variable.

---

# 2. What is an Array?

An array is a collection of multiple values stored under a single variable name.

Think of an array as a row of boxes.

```
Index

0    1    2    3    4

+----+----+----+----+----+
| 10 | 25 | 30 | 45 | 50 |
+----+----+----+----+----+
```

Each value has an **index**.

JavaScript arrays start from **0**.

This is exactly the same indexing rule used in C++. :contentReference[oaicite:1]{index=1}

# 8. Traversing Arrays

Using a for loop

```javascript
let marks = [80,75,92,60];

for(let i=0; i<marks.length; i++){

    console.log(marks[i]);

}
```

Output

```
80
75
92
60
```

---

# 9. The for...of Loop

JavaScript provides a special loop for arrays.

```javascript
let marks = [80,75,92];

for(const mark of marks){

    console.log(mark);

}
```

Output

```
80
75
92
```

# 10. Useful Array Methods

## push()

Adds a new value.

```javascript
let numbers = [1,2,3];

numbers.push(4);
```

Result

```
[1,2,3,4]
```

---

## pop()

Removes the last value.

```javascript
numbers.pop();
```

---

## shift()

Removes the first value.

```javascript
numbers.shift();
```

---

## unshift()

Adds a value at the beginning.

```javascript
numbers.unshift(10);
```

# 11. Arrays as Function Parameters

Example

```javascript
function average(numbers){

    let sum = 0;

    for(const value of numbers){

        sum += value;

    }

    return sum / numbers.length;

}

let marks = [80,75,92];

console.log(average(marks));
```

Output

```
82.33
```

This follows the same idea as the C++ lecture, where an array is passed to a function to calculate its average. :contentReference[oaicite:2]{index=2}

# 12. Objects

In the C++ lecture, structures were introduced to group related data together.

JavaScript uses **Objects** for the same purpose.

Example

```javascript
let student = {

    rollNo : 101,

    name : "Ali",

    cgpa : 3.8

};
```

# 16. Enumerations (Enums)

C++ provides

```cpp
enum Gender{

male,

female

};
```

JavaScript does not have built-in enums.

Instead, we create constant objects.

```javascript
const Gender = Object.freeze({

    MALE : "Male",

    FEMALE : "Female"

});
```

Usage

```javascript
let gender = Gender.MALE;

console.log(gender);
```

Output

```
Male
```

# 17. Introduction to Classes

Objects are created from classes.

Example

```javascript
class Student{

    constructor(name,cgpa){

        this.name = name;

        this.cgpa = cgpa;

    }

}
```

Creating objects

```javascript
let s1 = new Student(

    "Ali",

    3.8

);

console.log(s1.name);
```

Output

```
Ali
```

# 18. Program Composition

Every program consists of two major parts.

## Code

Examples

- Variables
- Assignment
- Conditions
- Loops
- Function calls
- Output
- Input

These instructions tell the computer **what to do**.

---

## Data

Examples

- Numbers
- Strings
- Booleans
- Arrays
- Objects
- Classes

Data represents **the information that the program works on**.

This matches the idea presented in the C++ lecture, where a program is composed of **code** and **data**. :contentReference[oaicite:3]{index=3}

# 19. Common Mistakes

❌ Accessing an invalid index

```javascript
marks[20]
```

Returns

```
undefined
```

---

❌ Forgetting array indices start at zero

Wrong

```javascript
marks[1]
```

when trying to access the first element.

Correct

```javascript
marks[0]
```

---

❌ Trying to modify a frozen enum

```javascript
Gender.MALE = "Boy";
```

This does not work because the object is frozen.
