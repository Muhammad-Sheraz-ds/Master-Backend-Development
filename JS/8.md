# 1. Introduction

In programming, different data types often work together.

Sometimes JavaScript automatically converts one data type into another.

Sometimes the programmer performs the conversion manually.

This process is called **Type Conversion (Type Casting).**

Unlike C++, JavaScript is a **dynamically typed language**, so type conversion happens more frequently.

---

# 2. Example Problem

Consider the following program.

```javascript
let a = 63;

let b = 10;

let result = a / b;

console.log(result);
```

Output

```
6.3
```

Notice that JavaScript automatically performs floating-point division.

This is different from C++, where dividing two integers results in integer division unless an explicit cast is used. The original lecture uses this example to motivate type casting. :contentReference[oaicite:1]{index=1}

---

# 3. What is Type Conversion?

Type conversion means changing a value from one data type to another.

Example

```
"100"

↓

100
```

or

```
25

↓

"25"
```

There are two types of conversion:

- Implicit Type Conversion
- Explicit Type Conversion

---

# 4. Implicit Type Conversion

Implicit conversion happens automatically.

JavaScript performs it when necessary.

Example

```javascript
console.log("10" * 2);
```

Output

```
20
```

JavaScript converts

```
"10"

↓

10
```

before multiplication.

---

Another example

```javascript
console.log(5 + "5");
```

Output

```
55
```

The number is converted into a string.

```
5

↓

"5"
```

Result

```
"55"
```

---

# 5. More Examples of Implicit Conversion

```javascript
console.log(true + 1);
```

Output

```
2
```

Because

```
true

↓

1
```

---

```javascript
console.log(false + 10);
```

Output

```
10
```

Because

```
false

↓

0
```

---

# 6. Explicit Type Conversion

Explicit conversion is performed by the programmer.

JavaScript provides several built-in conversion functions.

- Number()
- String()
- Boolean()
- parseInt()
- parseFloat()

This serves the same purpose as explicit casting in the C++ lecture, although JavaScript uses functions instead of cast operators. :contentReference[oaicite:2]{index=2}

---

# 7. Converting to Number

Syntax

```javascript
Number(value)
```

Example

```javascript
let age = "20";

let numberAge = Number(age);

console.log(numberAge);
```

Output

```
20
```

Type

```javascript
console.log(typeof numberAge);
```

Output

```
number
```

---

# 8. parseInt()

Used when converting integers.

```javascript
console.log(parseInt("45"));
```

Output

```
45
```

Example

```javascript
console.log(parseInt("45px"));
```

Output

```
45
```

---

# 9. parseFloat()

```javascript
console.log(parseFloat("3.14"));
```

Output

```
3.14
```

---

# 10. Converting to String

Syntax

```javascript
String(value)
```

Example

```javascript
let marks = 90;

let text = String(marks);

console.log(text);
```

Output

```
90
```

Type

```
string
```

---

# 11. Converting to Boolean

```javascript
Boolean(1)
```

Output

```
true
```

```javascript
Boolean(0)
```

Output

```
false
```

Examples

| Value | Boolean Result |
|---------|----------------|
| 1 | true |
| 0 | false |
| "" | false |
| "Hello" | true |
| null | false |
| undefined | false |

# 12. The typeof Operator

JavaScript provides

```javascript
typeof
```

to determine a variable's type.

Example

```javascript
let x = 100;

console.log(typeof x);
```

Output

```
number
```

Example

```javascript
console.log(typeof "Ali");
```

Output

```
string
```

---

# 13. Primitive Data Types

JavaScript provides these primitive types:

| Type | Example |
|------|---------|
| Number | 25 |
| String | "Ali" |
| Boolean | true |
| Undefined | undefined |
| Null | null |
| BigInt | 123n |
| Symbol | Symbol() |

Unlike the C++ lecture, JavaScript does not expose memory sizes such as `int`, `float`, or `double`; it uses a single `Number` type for most numeric values. :contentReference[oaicite:3]{index=3}

---

# 14. Comparison with C++

| C++ | JavaScript |
|------|------------|
| int | Number |
| float | Number |
| double | Number |
| char | String |
| bool | Boolean |

JavaScript combines several C++ numeric types into one **Number** type.

# 15. Common Mistakes

❌

```javascript
console.log("10" + 5);
```

Output

```
105
```

Correct

```javascript
console.log(Number("10") + 5);
```

Output

```
15
```

---

❌

```javascript
Number("ABC");
```

Output

```
NaN
```

Always check whether user input is valid.

---

# 16. Best Practices

✔ Use `Number()` when reading numeric input.

✔ Prefer `===` instead of `==`.

✔ Avoid relying on automatic type conversion unless it is intentional.

✔ Use `typeof` while debugging.
