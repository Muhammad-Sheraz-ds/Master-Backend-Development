# 1. Why Do We Need Decision Making?

In programming, a program often needs to make decisions based on certain conditions.

Example:

- If marks are greater than or equal to 50 → Display "Pass"
- Otherwise → Display "Fail"

Without decision-making statements, every instruction would execute regardless of the situation.

# 2. Comparison Operators

Comparison operators compare two values and return either `true` or `false`.

| Operator | Meaning | Example |
|----------|---------|---------|
| == | Equal to | `a == b` |
| === | Strict Equal | `a === b` |
| != | Not Equal | `a != b` |
| > | Greater Than | `a > b` |
| < | Less Than | `a < b` |
| >= | Greater Than or Equal | `a >= b` |
| <= | Less Than or Equal | `a <= b` |

Example

```javascript
let age = 20;

console.log(age >= 18);
```

Output

```
true
```

---

# 3. Logical Operators

Sometimes we need to check multiple conditions together.

| Operator | Meaning |
|-----------|---------|
| && | AND |
| \|\| | OR |
| ! | NOT |

Example

```javascript
let age = 20;
let citizen = true;

console.log(age >= 18 && citizen);
```

Output

```
true
```

---

# 4. The `if` Statement

The `if` statement executes a block of code only when a condition is true.

Syntax

```javascript
if (condition) {
    statements;
}
```

Example

```javascript
let marks = 75;

if (marks >= 50) {
    console.log("Pass");
}
```

Output

```
Pass
```

---

# 5. The `if...else` Statement

When there are two possible outcomes, use `if...else`.

Syntax

```javascript
if (condition) {
    statements;
} else {
    statements;
}
```

Example

```javascript
let marks = 45;

if (marks >= 50) {
    console.log("Pass");
} else {
    console.log("Fail");
}
```

Output

```
Fail
```

---

# 6. The `else if` Ladder

When there are more than two conditions, use `else if`.

Example

```javascript
let marks = 83;

if (marks >= 90) {
    console.log("Grade A+");
}
else if (marks >= 80) {
    console.log("Grade A");
}
else if (marks >= 70) {
    console.log("Grade B");
}
else {
    console.log("Need Improvement");
}
```

Output

```
Grade A
```

---

# 7. The `switch` Statement

Use `switch` when checking one variable against multiple values.

Syntax

```javascript
switch (expression) {

    case value1:
        statements;
        break;

    case value2:
        statements;
        break;

    default:
        statements;
}
```

Example

```javascript
let day = 3;

switch(day){

    case 1:
        console.log("Monday");
        break;

    case 2:
        console.log("Tuesday");
        break;

    case 3:
        console.log("Wednesday");
        break;

    default:
        console.log("Invalid Day");
}
```

Output

```
Wednesday
```

# 8. Introduction to Loops

Suppose you want to print

```
Hello
```

10 times.

Without loops:

```javascript
console.log("Hello");
console.log("Hello");
console.log("Hello");
...
```

This is repetitive.

Loops solve this problem.

---

# 9. The `while` Loop

The `while` loop checks the condition before executing the loop body.

Syntax

```javascript
while(condition){

    statements;

}
```

Example

```javascript
let i = 1;

while(i <= 5){

    console.log(i);

    i++;
}
```

Output

```
1
2
3
4
5
```

---

# 10. The `do...while` Loop

The `do...while` loop executes at least once.

Syntax

```javascript
do{

    statements;

}while(condition);
```

Example

```javascript
let i = 1;

do{

    console.log(i);

    i++;

}while(i <= 5);
```

Output

```
1
2
3
4
5
```

Difference from `while`:

- `while` may execute zero times.
- `do...while` executes at least one time.

---

# 11. The `for` Loop

The `for` loop is commonly used when the number of iterations is known.

Syntax

```javascript
for(initialization; condition; update){

    statements;

}
```

Example

```javascript
for(let i = 1; i <= 5; i++){

    console.log(i);

}
```

Output

```
1
2
3
4
5
```

---

# 12. Choosing the Right Loop

| Loop | Use When |
|-------|----------|
| while | Number of iterations is unknown |
| do...while | Loop must execute at least once |
| for | Number of iterations is known |

# 13. Functions

A function is a reusable block of code that performs a specific task.

Instead of writing the same code repeatedly, we place it inside a function.

---

# 14. Declaring a Function

Syntax

```javascript
function functionName(parameters){

    statements;

}
```

Example

```javascript
function greet(){

    console.log("Welcome!");

}
```

Calling the function

```javascript
greet();
```

Output

```
Welcome!
```

---

# 15. Function with Parameters

Example

```javascript
function greet(name){

    console.log("Hello " + name);

}

greet("Ali");
```

Output

```
Hello Ali
```

---

# 16. Returning Values

Functions can return values.

Example

```javascript
function average(a, b, c){

    return (a + b + c) / 3;

}

let result = average(10,20,30);

console.log(result);
```

Output

```
20
```

# 17. Arrow Functions (Modern JavaScript)

JavaScript also provides a shorter syntax called an arrow function.

```javascript
const average = (a, b, c) => {

    return (a + b + c) / 3;

};
```

Short version

```javascript
const square = x => x * x;
```

# Common Mistakes

❌ Forgetting braces

```javascript
if(age > 18)
console.log("Adult");
```

✔ Better

```javascript
if(age > 18){

    console.log("Adult");

}
```

---

❌ Infinite Loop

```javascript
let i = 1;

while(i <= 5){

    console.log(i);

}
```

Why?

Because `i` never changes.

Correct

```javascript
i++;
```

---

❌ Forgetting `return`

```javascript
function add(a,b){

    a+b;

}
```

Correct

```javascript
function add(a,b){

    return a+b;

}
```
